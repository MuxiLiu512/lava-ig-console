# 影像品質檢核清單（自動＋人工）

> 迭代 harness 可新增檢查點（低風險，自動生效）。

## 自動品檢（生圖後）
- [ ] 逐人數：手臂/手/手指數量正確，無多指少指融指
- [ ] 臉部：五官對稱、無扭曲、眼睛無異常
- [ ] 無穿模、無多餘肢體
- [ ] 無倒影穿幫（鏡面/玻璃）
- [ ] 無文字浮水印/簽名/logo
- [ ] 缺陷 → 自動重生一次

## 色調品檢（整篇）
- [ ] 亮色調 ≥ 2 張
- [ ] 非同一色溫（避免六張同暖同暗）
- [ ] 琥珀感 ≤ 2 張

## 排版品檢（渲染後）
- [ ] display_copy 未被腰斬、英文整字換行
- [ ] 資料來源行已縮小轉灰置段末
- [ ] 【重點黃】〖超級重點橘〗數量合規（〖〗一篇 ≤ 2 處）
- [ ] footer 固定、logo 可讀（top scrim 生效）

## 人工複核（操控室）
- [ ] 封面 hook 是全篇最強問句
- [ ] 底圖與文案語義相符
- [ ] 劇照出處已標注
