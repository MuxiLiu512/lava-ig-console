# 風格規格 — 機器可改增補層

> 正本規格 Doc 在 Google Drive（v1.0 / v1.1），**不由機器修改**。
> 本檔是迭代 harness 可安全增修的「增補層」，渲染與生圖以「正本 + 本增補」為準。
> 每次修改 = 一個 git commit（cfg-NNN），並寫入 data/iterate_log.json。

## §光線（硬規則）
- 六張中亮色調至少 **2** 張；禁全暖暗；琥珀感最多 2 張。
- dark = 冷色藍調時刻，禁漆黑。

## §Hook
- 整篇最強 hook 問句必須上第 1 張，不得埋在內頁結尾。
- 每張 display_copy 上限約 140 字。

## §選圖
- 優先序：真實感 > 精緻感。首選電影/影劇劇照與迷因（標注出處）；次選 Pinterest 參考後 seedream 生成。
- 同編號多檔時：劇照/迷因（.jpg/.gif）自動優先於生成圖（.png）。

## §CTA（逐字鎖定，機器不得改主文案）
Lava 不聊天的交友軟體／滑動、配對、碰面／Lava 為你選好地點，現場備有破冰卡牌與迎賓飲品／出現就好，其他交給我們。

## §語氣
雜誌感、有溫度、有觀點；每段至少一長句；短句只收尾；繁中，技術詞可英文。

## §絕對禁止句式
「不是…而是…」／破折號 —— ／短斷句連發／赦免式安慰。（CTA 尾板豁免）

## §畫布比例（07-15 定案）
- 一律輸出 **4:5（1950×2438）**，對齊 IG 動態滿版上限（1080×1350）。禁用舊 3:4（2600）——會被 IG 上下裁掉 logo 與 footer。
- 重要元素（頂部 logo／eyebrow、底部 footer/CTA）務必落在畫布內；渲染引擎 render_post_v5 已內建。
